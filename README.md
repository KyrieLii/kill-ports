# kill-ports

A simple script to kill ports.

Only used in Linux(MacOS).

## Install

```shell
npm i -g  kill-all-ports

kill-ports 3000
```

or

```shell
npx kill-all-ports 3000
```

## Usage

```shell
kill-ports 3000
```

or kill ports list

```shell
kill-ports 3000 4000 5000
```

or kill range ports

```shell
kill-ports 3000...3010
```
